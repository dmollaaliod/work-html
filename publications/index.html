Note on formatting

Some of the older papers have been formatted in postscript, and compressed with the gzip compressing tool. I think that the web server over here automatically delivers the files uncompressed, but I'm not sure about it now. If you need to uncompress the files, you can download gzip. The resulting postscript file can be printed directly on any postscript printer. If you want to preview the files, you can always use ghostscript.

Papers formatted in PDF are not compressed (PDF contains a compression algorithm). A popular free previewer of PDF files is Acrobat Reader.

If you have trouble in downloading any of the papers, or you cannot uncompress/print/view them, I will be happy to send you the papers in your preferred format (that is, if I can do it!). Just send me an e-mail.
